A simple GUI-based Silent Auction application built using Python and Tkinter.
The application allows multiple users to enter bids, maintains bid history, and determines the highest bidder at the end of the auction.

Features:

-User-friendly GUI interface using Tkinter

-Accepts only integer values for bid amounts

-Stores bidder names and bids dynamically

-Displays live bid history

-Determines and displays the highest bidder

-Separate result window for auction outcome

How It Works:

-User enters Bidder Name and Bid Amount

-Bid amount is validated to allow only numeric input

-Submitted bids are stored in a dictionary

-Bid history updates in real time

-User chooses whether to add more bidders

-When auction ends, the highest bid is calculated

-Winner is displayed in a separate window
